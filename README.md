## 基于知识图谱的旅游推荐问答系统

### 项目简介

本项目是一个基于知识图谱的旅游推荐问答系统，它结合了爬虫技术、图数据库Neo4j、自然语言处理以及推荐算法，为用户提供个性化的旅游景点推荐和问答服务。

### 主要功能

- 景点数据爬取和存储
- 基于Neo4j的旅游知识图谱构建
- 用户登录注册和信息管理
- 景点列表展示和详细信息查看
- 景点评分评论和协同过滤推荐
- 自然语言问答和知识图谱展示


### 技术架构

- **前端:** 基于React开发，提供用户界面和用户交互功能。
- **后端:** 基于Django框架提供RESTful API接口。
- **数据库:** Neo4j图数据库存储旅游知识图谱。
- **工具:** 爬虫技术、jieba分词等自然语言处理工具、ECharts可视化库。
